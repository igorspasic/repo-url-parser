# VCS URL PARSER

![](https://img.shields.io/maven-central/v/com.oblac/repo-url-parser.svg)

Parses various VCS URLs:

+ Github
+ Github Enterprise

TODO:

+ Bitbucket

## Get it

With Gradle:

	compile 'com.oblac:repo-url-parser:<version>'

## Usage

It's complicated.

	GithubRepo gh = GithubUrl.gh("https://github.com/oblac/jodd");

That's it :)
